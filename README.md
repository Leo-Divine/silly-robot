# silly-robot

## Temporary Command Documentation:
0 - Runs following code;
1 - Stops all code;

## Temporary Block Documentation:
000 - SetSpeed(Speed);
001 - Move(Distance);
002 - TurnLeft();
003 - TurnRight();
004 - Rotate(Degrees);

005 - SetLeftColor(Color);
006 - SetRightColor(Color);
007 - SetColor(Color);

008 - Wait(Seconds);
009 - Loop(CodeLineCount, LoopTimes);

## Color Documentation:
0 - Red
1 - Orange
2 - Yellow
3 - Green
4 - Blue
5 - Purple
6 - White
